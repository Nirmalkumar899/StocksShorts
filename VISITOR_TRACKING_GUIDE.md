# Complete Visitor Tracking Guide for StocksShorts

## How to Check Visitor Numbers

### 1. Real-Time Visitor Counter (Available Now)
Your website now shows a live visitor counter in the bottom-right corner:
- **Daily Visitors**: Total unique visits today
- **Session Visitors**: Current browser session count

### 2. Google Analytics Setup (Detailed Tracking)

**Step 1: Create Google Analytics Account**
1. Go to [Google Analytics](https://analytics.google.com/)
2. Click "Start measuring"
3. Create account for "StocksShorts"
4. Choose "Web" platform
5. Enter website URL: `https://stocksshorts.com`

**Step 2: Get Measurement ID**
1. Copy your Measurement ID (format: G-XXXXXXXXXX)
2. Add it as environment variable in Replit:
   - Go to Secrets tab
   - Add: `VITE_GA_MEASUREMENT_ID` = `G-XXXXXXXXXX`

**Step 3: Deploy Changes**
Once you add the measurement ID, your site will start tracking:
- Page views
- User sessions
- Traffic sources
- User behavior
- Conversion events
